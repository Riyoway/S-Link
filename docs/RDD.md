## 🌟 2. プロジェクト概要
**プロジェクト名:** 実用的で利便性を重視したアシスタントWebアプリ  
**プロジェクトマネージャー:** TBD  
**プロジェクトスポンサー:** TBD  
**ステークホルダー:** 生徒、教員、学校管理者  

**目的:**  
学校生活における情報確認・課題管理の不便さを解消する利便性重視のWebアプリを提供する。

**範囲:**  
- シラバス・時間割・課題・バス時刻・天気情報・教室場所の確認  
- 外部連携（Discord, LINE）  
- 成績シミュレーション機能  
- PWA対応でスマホ・タブレットでも利用可能  

**目標:**  
- 情報確認の効率化  
- 課題進捗管理の改善  
- 通学・校内移動の利便性向上

## 📚 3. 用語集 / 定義
| 用語 | 定義 |
|------|------|
| PWA | プログレッシブWebアプリ：ブラウザで動作し、スマホにインストール可能 |
| Classroom API | Google Classroom API：課題やシラバス取得用 |
| Weather API | Weather News API：天気情報取得用 |

## ⚡ 4. 機能要件
| ID  | 要件内容 | 優先度 | 備考 |
|-----|----------|-------|------|
| FR-1 | シラバスと学校行事の表示 | 高 | 視覚的にわかりやすく表示 |
| FR-2 | 学年・コース別の時間割表示 | 高 | 日次更新、自動切替 |
| FR-3 | バス時刻表・混雑予測表示 | 高 | 遅延情報も反映 |
| FR-4 | 課題一覧・期限管理 | 高 | 通知機能付き |
| FR-5 | 教室の場所表示 | 中 | 校内マップ連携 |
| FR-6 | 天気情報表示 | 中 | API連携 |
| FR-7 | 外部アプリ連携（Discord, LINE） | 低 | 通知・チャット連携 |
| FR-8 | 成績シミュレーション機能 | 低 | 教員入力ベース |
| FR-9 | 初期設定（学年・コース・寮生） | 高 | 初回利用時設定 |
| FR-10 | ホーム画面：お知らせ＆今日の予定表示 | 高 | 翌日自動切替 |

## 💾 8. データ要件
| データエンティティ | 属性 | 備考 |
|------------------|------|------|
| ユーザー | id, 名前, メール, 学年, コース, 寮フラグ | 初期設定情報 |
| 時間割 | 日付, 時間, 科目, 教室 | 日次更新、自動切替 |
| 課題 | id, タイトル, 期限, 状態 | 完了/未完了管理 |
| バス | route_id, 時刻, 混雑度 | API取得 |
| 天気 | 日付, 気温, 天気 | API連携 |
